Responsive Redesign Project

Project Overview

CSCI1300: User Interface and User Experience - Responsive Redesign

The goal of the project was to chose a website we believed was in need of a little revamping... The purpose was to evaluate the site in its current state, then design and deploy a new and improved responsive redesign. I chose to redesign a local Providence nail salon, Metro Nails.

Usability Evaluations

Learnability

  • The site's learnability is decent. After one or two uses the user may get the hang of it.
  • The home page lacks structure and organization.
  • The navbar contains phrasing that may be confusing, for example "menu" should be "services".
  • The location and hours are lost amongst a block of text that is poorly formatted.
  • Users would need to scroll and search amongst text for annoucements and any relevant information.

Memorability

  • The site's memorability is good.
  • Since the site is relatively simple and mainly used to display locations, hours, prices, and services there is not much for users to forget.
  • The user would only need to learn to scroll down to view any important information in the text wall.

Accessibility

To evaluate the sites accessibility I put Metro Nails into WAIVE, an accessibility evaluation software. Several errors came up.

A summary of these errors include:

  • Low Contrast Errors - All contrast errors are predominantly in the nav bar and on the homepage. The site is grey with darker grey or black overlayed text that is not properly spaced or formatted making it very difficult to read.
  • Justified Text Errors - The site consists of large blocks of justified text that make it difficult to read.
  • Alternative Text Errors - All images do not contain an alt key tag making it inaccessible for those that need a screen reader. Without the tag images would not be available to those users.
